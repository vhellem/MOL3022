##  Running. 

### Backend: 

In the backend folder. 

```
python3 -m venv venv
```

```
source venv/binb/activate
```

```
pip install -r requirements.txt
```

```
python app.py
```

### Frontend: 

With yarn installed. (npm install -g yarn) 

```
yarn install
```

```
yarn
```
